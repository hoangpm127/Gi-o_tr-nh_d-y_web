Web Builder Foundation

Giáo trình 5 buổi — giúp người Việt chưa biết gì về code tự build được 1 web app chắc lõi, sạch code, bằng cách ra lệnh AI đúng cách.

Triết lý xuyên suốt

1
Tư duy hệ thống trước, code sau. Mỗi buổi mở đầu bằng 1 mô hình tư duy — cái khung để hiểu tại sao mọi thứ hoạt động. Không học syntax thuộc lòng.
2
Ra lệnh AI ngày càng chuẩn hơn. Mỗi buổi nâng 1 bậc kỹ năng viết yêu cầu cho AI — từ mô tả giao diện → mô tả hành vi → mô tả lỗi → mô tả cải tiến.
3
Biết tìm và mô tả lỗi — không chỉ sao chép code. Người học hiểu đủ để nói chính xác "cái gì đang xảy ra, cái gì mong muốn" — AI sẽ fix đúng.
4
Khung xương vững trước khi trang trí. HTML là bộ xương → CSS là quần áo → JS là hành động → Data là trí nhớ → Deploy là ra đời. Đúng thứ tự.
5
Tiếng Việt 100%. Mọi khái niệm đều có tên tiếng Việt dễ hiểu. Thuật ngữ kỹ thuật chỉ giới thiệu sau khi đã hiểu bằng ngôn ngữ tự nhiên.

Đưa lên mạng — Ra đời
Deploy · URL thật · Người khác truy cập được · 2 tiếng
Chưa build
Mô hình tư duy buổi này
Localhost vs Internet. Máy bạn là "nhà riêng" — chỉ bạn vào được. Server là "nhà công cộng" — ai cũng vào được. Deploy là chuyển file từ nhà riêng lên nhà công cộng.
